# graphql-apollo-react-setup

This is GraphQL - Apollo - React test setup

To run this application, execute the following commands:

  1. Install npm modules

    ```
    $ npm install
    ```

  2. Start client side server:

    ```
    $ npm run start
    ```

  3. Start back-end server

    ```
    $ npm run dev
    ```

  4. or start both

    ```
    $ npm run all
    ```
